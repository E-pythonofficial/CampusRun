# CampusRun
Fast. Secure. Campus Powered. Student-to-student campus delivery Live tracking • Secure PIN • Reliable dispatch Built for smarter campus logistics


Campus Run is a modern, student-led errand and delivery platform designed to eliminate the daily logistics "stress" within university environments. By connecting students who need items moved with peer "Runners" already on those routes, we create a high-speed, low-cost network that benefits the entire campus community.

🚀 Mission: Less Pay. Do Less. No Stress.
We believe campus life is demanding enough without worrying about how to get a package, a meal, or an essential item from one side of the university to the other. Campus Run is built to ensure students can focus on their goals while we handle the movement.

✨ Key Features
30 Mins Max: Leveraging student proximity to ensure deliveries happen faster than traditional services.

Escrow Secure: Integrated payment protection where your parcel is only released via a unique 4-digit Handshake PIN.

Item Capture: Digital photo verification at pickup to ensure the safety and security of every valuable moved.

Flat-Fee Pricing: Transparent ₦800 pricing for standard runs, with a cap of ₦1,300 for high-value items.

🛠️ Tech Stack
This project is a full-stack web application focused on high performance and smooth UI/UX:

Frontend: React with TypeScript for a robust, type-safe codebase.

Styling: Tailwind CSS for a modern, responsive "startup" aesthetic.

Animations: Framer Motion for high-end staggered entrances and interactive transitions.

Mapping: Leaflet and React-Leaflet for real-time coordination and location tracking.

Icons: Lucide React for clean, minimalist visual language.

🏗️ Project Structure
The repository is partitioned into distinct roles to handle the complex ecosystem of campus logistics:

Requester: For students needing errands run.

Dispatcher: To manage and assign active runs.
